Pokemon battle

Game Description
This is a game where people on different computers can log in to the game, and battle each other in a Pokemon style environment.

How to play

  • Make a username using the navbar link
  • Click on "Ready to battle" and select your username from the dropdown
  • Click on the "New Game" button to create a room
  • Use the unique room id to allow a friend to connect to the room
  • Click attack to start the battle
